시작하기

학원장 가입

keytrend 학원 관리 시스템은 학원장 계정으로 이용할 수 있습니다.

  1. www.keytrend.co.kr 에서 회원가입(/signup) 페이지로 이동합니다.
  2. 상단 탭에서 「학원장」 탭을 클릭합니다.
  3. 이름, 학원 이름, 이메일, 비밀번호, 비밀번호 확인을 입력합니다.
  4. 약관 동의: (필수) 이용약관, (필수) 개인정보처리방침, (선택) 마케팅 정보 수신에 체크합니다. 「모두 동의합니다」로 한 번에 체크할 수 있습니다.
  5. 「회원가입」 버튼을 클릭합니다.
  6. "입력하신 이메일로 인증 메일을 보냈습니다"라는 안내가 표시됩니다. 메일함에서 noreply@keytrend.co.kr이 보낸 메일을 열고 인증 링크를 클릭하면 "이메일 인증 완료" 화면이 표시되고, 로그인 후 이용할 수 있습니다.
💡 팁 인증 메일이 보이지 않을 때 — 스팸함(정크 메일함)을 먼저 확인해 주세요. 그래도 없다면 keytrend777@gmail.com 으로 문의해 주시기 바랍니다.
💡 팁 학원장 탭 아래의 「학원전용 온라인 클래스 개설 및 운영 안내서 보기」 링크를 누르면 가입 전 전체 운영 흐름을 4단계로 미리 볼 수 있습니다.

로그인과 첫 화면

가입 완료 후 로그인하면 학원 관리 페이지가 열립니다. 화면은 위에서 아래로 스크롤하는 구조이며, 처음에는 다음 세 부분이 보입니다.

  1. 학원 정보 — 학원명과 학원장 정보가 표시됩니다.
  2. 학원 전체 학원생 — 학원에 소속된 모든 학생 목록입니다. 처음에는 접혀 있으며, 클릭하면 펼쳐집니다.
  3. 반 관리 — 반을 만들고 목록을 확인하는 곳입니다.
⭐ 중요 가장 중요한 사용 원칙 — 학생 초대, 숙제 등록, 결과 조회 등 대부분의 기능은 반을 먼저 선택해야 나타납니다. 「반 관리」의 내 반 목록에서 반 하나를 클릭하면 화면 아래에 반 상세 영역이 열리고, 그 안에서 초대·숙제·리포트 기능을 사용하게 됩니다. 아직 반이 없다면 2장의 안내에 따라 반부터 만들어 주세요.

반 만들기와 관리

반 만들기

반은 학생을 묶어 숙제를 내고 성적을 관리하는 기본 단위입니다. 학원 → 반 → 학생의 3단계 구조로 운영됩니다.

  1. 「반 관리」 영역의 입력창에 반 이름을 입력합니다. (예: 고1 A반, 수능 집중반)
  2. 「반 만들기」 버튼을 클릭합니다.
  3. 만들어진 반은 바로 아래 내 반 목록에 나타납니다.
💡 팁 반 이름 정하기 — 학년·요일·수준이 드러나게 지으면 반이 많아져도 찾기 쉽습니다. (예: 고2 화목 심화반)
⚠️ 주의 반 이름은 만든 후 수정할 수 없습니다. 이름을 바꾸려면 반을 삭제하고 새로 만들어야 하며, 반 삭제 시 그 반의 학생 배정과 숙제 데이터가 모두 삭제되므로 처음에 신중히 정해 주세요. (FAQ Q7 참조)

반 배정 — 한 학생을 여러 반에서 관리하기

이미 학원에 소속된 학생을 다른 반에도 배정할 수 있습니다. 예를 들어 고1 A반 학생을 어휘 특강반에도 넣어 두 반의 숙제를 모두 받게 하는 방식입니다.

  1. 배정받을 반(예: 어휘 특강반)을 내 반 목록에서 클릭해 반 상세를 엽니다.
  2. 「반 배정」 제목을 클릭해 펼칩니다.
  3. 검색창에서 학생을 이름·이메일·현재 반으로 검색합니다.
  4. 학생 옆의 「이 반에 배정」 버튼을 누르고, 확인창에서 확인하면 배정이 완료됩니다.
💡 팁 배정이 완료되면 해당 학생에게 알림 이메일이 자동 발송됩니다. 별도로 연락하지 않으셔도 됩니다.

학원생 초대하기

초대 전 필수 확인: 사전 동의 체크박스

반 상세 맨 위에는 주황색 필수 동의 확인 박스가 있습니다.

☑ (필수) 학원생 또는 보호자에게 keytrend 가입 초대 발송에 대한 사전 동의를 받았습니다.

초대 이메일은 학생(미성년자의 경우 보호자)의 동의 없이 발송할 수 없습니다. 이는 개인정보 보호법을 준수하기 위한 절차로, 초대 전에 반드시 학생 또는 보호자에게 구두·서면 등으로 동의를 받으신 후 체크박스를 체크해 주세요. 체크하지 않으면 개별 초대와 일괄 초대 모두 진행할 수 없습니다.

개별 초대

한 명씩 초대할 때 사용합니다.

  1. 반 상세의 「학원생 초대 (개별)」 영역으로 이동합니다.
  2. 이름(선택 사항)과 초대할 이메일을 입력합니다.
  3. 권장 플랜을 선택합니다(선택 사항). 권장 플랜이 무엇인지는 3.5를 참고해 주세요.
  4. 「초대 보내기」 버튼을 클릭합니다.

초대 메일이 학생에게 발송되고, 학생이 메일의 링크로 가입하면 자동으로 이 반에 소속됩니다.

엑셀/CSV 일괄 초대

여러 명을 한 번에 초대할 때 사용합니다.

파일 준비

  • 엑셀(.xlsx, .xls) 또는 CSV 파일
  • 첫 번째 열에 이름, 두 번째 열에 이메일을 입력합니다.
  • 한 번에 최대 50명까지 업로드할 수 있습니다.

업로드와 전송

  1. 「학원생 초대 (엑셀/CSV 일괄)」 영역에서 파일을 업로드합니다.
  2. 업로드하면 명단 미리보기가 표시됩니다. 이름·이메일이 올바른지 확인해 주세요.
  3. 필요하면 전체 권장 플랜을 선택합니다(명단 전원에게 일괄 적용).
  4. 「전체 초대 전송」 버튼을 클릭합니다. 잘못 올렸다면 「취소」를 누르고 다시 업로드하면 됩니다.

초대 발송 이력 확인

「초대 발송 이력」 제목을 클릭해 펼치면 지금까지 보낸 초대를 확인할 수 있습니다.

  • 검색: 이메일 또는 이름으로 찾을 수 있습니다.
  • ↻ 새로고침: 최신 상태로 갱신합니다.
  • 표에는 수신자 / 발송일 / 상태 / 추천 플랜 / 동의 확인이 표시됩니다.

초대를 받았지만 아직 가입하지 않은 학생은 「학원 전체 학원생」의 예비 학원생 (가입 대기) 목록에서도 확인할 수 있습니다. 가입이 완료되면 자동으로 학원생 목록으로 옮겨집니다.

권장 플랜이란?

초대할 때 학생에게 추천할 구독 플랜을 지정하는 기능입니다. 지정하면 학생이 가입할 때 해당 플랜이 추천으로 표시되어, 학원의 학습 계획에 맞는 플랜을 학생이 쉽게 선택할 수 있습니다.

플랜은 도전 / 합격 / 만점 세 가지가 있으며, 요금과 플랜별 혜택은 요금 안내 페이지에서 확인해 주세요.

💡 팁 권장 플랜은 말 그대로 추천입니다. 학생이 다른 플랜을 선택하거나 무료로 시작하는 것도 가능합니다. 어느 플랜을 추천할지는 반의 학습량(챗봇 질문 횟수 등)을 기준으로 정하시면 됩니다.

기존 회원 직접 추가와 열람 동의

기존 keytrend 회원 직접 추가

이미 keytrend에 가입되어 있는 학생은 초대 메일 없이 바로 학원에 추가할 수 있습니다.

  1. 「학원 전체 학원생」 제목을 클릭해 펼칩니다.
  2. 「기존 회원 직접 추가」 영역의 입력창에 학생의 가입 이메일을 입력합니다.
  3. 「검색」 버튼을 클릭합니다.
  4. 검색된 회원 옆의 「학원에 추가」 버튼을 누르면 바로 추가됩니다(별도 확인창 없음).
💡 팁 초대와 직접 추가의 차이 — 초대(3장)는 아직 keytrend 회원이 아닌 학생에게 가입 링크를 보내는 것이고, 직접 추가는 이미 가입된 회원을 학원 명단에 넣는 것입니다. 검색한 이메일이 미가입 상태라면 버튼이 「예비 학원생으로 추가」로 바뀌며, 이 경우 예비 학원생 명단에 등록됩니다.

열람 동의가 필요한 이유

직접 추가된 학생의 학습 현황(정답률, 학습 통계, 약점 분석 등)은 개인정보이므로, 학생 본인의 동의가 있어야 학원장이 열람할 수 있습니다.

학생을 직접 추가하면 학생에게 동의 요청 이메일이 자동 발송됩니다. 학생이 메일 또는 사이트의 안내에 따라 동의하면 그때부터 학습 현황이 표시됩니다.

동의 대기 상태와 재발송

아직 동의하지 않은 학생은 학원생 목록에서 주황색 동의 대기 배지로 표시됩니다.

  • 학생이 동의 메일을 놓쳤다면 해당 학생 행의 「동의 요청 재발송」 버튼으로 다시 보낼 수 있습니다.
  • 「학원 전체 학원생」 제목 옆의 총원 표시에서 동의 대기 N명을 한눈에 확인할 수 있습니다.
